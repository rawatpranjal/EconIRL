\section{Reproducing every figure and table} \label{app:reproducing}

Every numbered figure and table in this paper is produced by exactly one script. To regenerate the paper end to end, run \code{bash papers/econirl\_package\_jss/run\_all.sh} from the repository root. All scripts use a fixed random seed of 42 and write their output to the location named in the third column. The orchestrator writes a manifest of every produced output to \code{manifest.csv} so the LaTeX build can verify that nothing was skipped.

The diagnostic scripts D1 and D2 emit feature-matrix rank, condition number, state coverage, and single-action state count to \code{diagnostics.csv}. Diagnostic failures abort the corresponding benchmark. The transfer scripts TT1 through TT5 perturb each Rust bus transition probability by a Beta(2, 2) noise variable, refit on the original panel, and report the percent optimal value of the recovered reward under the held-out transitions in \code{transfer\_results.csv}.
