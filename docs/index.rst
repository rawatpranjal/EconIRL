econirl
=======

EconIRL documents estimator validation on synthetic dynamic discrete choice
DGPs with known truth. The public RTD surface is intentionally lean: estimator
pages only, with reference PDFs for the estimator-specific math, tutorial,
known-truth recovery results, and counterfactual checks.

No public RTD page runs real-data estimation. Historical examples, tutorials,
API pages, dataset notes, and old planning material are archived in the
repository but excluded from the RTD build.
