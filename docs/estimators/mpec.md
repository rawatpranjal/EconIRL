# MPEC

**Reference PDF:** pending known-truth migration.

MPEC estimates the same structural likelihood as NFXP but treats the Bellman
fixed point as an explicit equality constraint. It is part of the 12-estimator
known-truth validation suite.

This RTD page is intentionally short while the dedicated MPEC TeX/PDF tutorial
is migrated to the shared synthetic DGP. No real-data estimation examples are
published on RTD.
