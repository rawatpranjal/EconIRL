# MCE-IRL

**Reference PDF:** pending known-truth migration.

MCE-IRL learns reward parameters by matching expert feature expectations under
the maximum causal entropy policy. It is part of the 12-estimator known-truth
validation suite.

This RTD page is intentionally short while the dedicated MCE-IRL TeX/PDF
tutorial is migrated to the shared synthetic DGP. No real-data estimation
examples are published on RTD.
