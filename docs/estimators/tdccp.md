# TD-CCP

**Reference PDF:** pending known-truth migration.

TD-CCP estimates dynamic discrete choice models through temporal-difference
equations for CCP value components, avoiding explicit transition-density
estimation. It is part of the 12-estimator known-truth validation suite.

This RTD page is intentionally short while the dedicated TD-CCP TeX/PDF
tutorial is migrated to the shared synthetic DGP. No real-data estimation
examples are published on RTD.
