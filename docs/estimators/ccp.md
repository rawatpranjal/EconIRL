# CCP

**Reference PDF:** pending known-truth migration.

CCP estimates structural dynamic discrete choice models by replacing the NFXP
inner Bellman solve with conditional-choice-probability inversion. It is part
of the 12-estimator known-truth validation suite.

This RTD page is intentionally short while the dedicated CCP TeX/PDF tutorial
is migrated to the shared synthetic DGP. No real-data estimation examples are
published on RTD.
