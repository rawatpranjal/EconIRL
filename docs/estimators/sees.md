# SEES

**Reference PDF:** pending known-truth migration.

SEES estimates structural dynamic discrete choice models by approximating the
value function with a sieve basis and penalizing Bellman residuals. It is part
of the 12-estimator known-truth validation suite.

This RTD page is intentionally short while the dedicated SEES TeX/PDF tutorial
is migrated to the shared synthetic DGP. No real-data estimation examples are
published on RTD.
